Ukraine - UKTZED

All products must be classified in process of their declaration. The codes are assigned to the goods in accordance with the classification groups specified in the UKTZED. There are no goods which are not supposed to be the subjects of classified. In the UKTZED, goods are systematized into sections, groups, commodity items, the names and numerical codes of which are unified with the Harmonized Commodity Description and Coding System. There are thousands of such codes.

That is why in this module you can independently choose four methods of updating / downloading data (from the module itself, from a Json File, through a standardized address of a specific resource on the Internet - URL) or manually.

For a convenient and quick search for the UKTZED code, this module also provides a search by both code and name, or by a specific group of products.

Module settings

To configure this module, select the Applications category, enter "uktz" in the search bar, and install the found module.

Next, go to Settings and activate developer mode. On the left, select Uktzed and click UPLOAD UKTZED.

After selecting this option, you get an access to three methods of downloading/updating codes. Choose the method you need and click Upload. In a few minutes, you will receive information about the codes in your Odoo system. You can find it in the general menu by selecting UKTZED. Also from this menu, you can create a new desired code by simply clicking the CREATE button.
