SMS API

This module overwrite odoo sms module, inherit sms.sms model for using local provider gates. Implements provider gateway connector installing.

Logging sms sends. Implements SandBox provider for testing sms without connecting to gates and sending real sms.

This is base module for sms provider integrations.

Integration module like kw_sms_turbosms is required for sending sms.

If you plan to create sms provider connectors, I recommend take bundle kw_mass_mailing_sms that includes current base module
